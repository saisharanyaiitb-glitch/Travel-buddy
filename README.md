# 🌍 Travel Buddy for Indian Tourists

A simple AI-powered web app that helps Indian travelers explore any city worldwide.

## 🚀 Features
- Search any city (Paris, Dubai, Bangkok, etc.)
- Find Indian restaurants & veg/Jain options
- Currency conversion & exchange tips
- Hotels & stay recommendations
- Transport guidance
- Safety tips & local insights
